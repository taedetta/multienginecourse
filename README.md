# Baron 58 Multi-Engine Course

Interactive study platform for the Beechcraft Baron 58 multi-engine rating course.

## Sections

- **Single Engine Aerodynamics** — Critical engine, PAST factors
- **VMC Table** — Control vs performance factors
- **BE58 Oral Review** — Systems, airspeeds, and checkride prep

## Activities (per section)

- Flash Cards
- Quick Quiz (10 random MC questions)
- Written Exam (full question bank, 80% to pass)
- Memory Match
- Section-specific mini-games (PAST Order, VMC Factor Match, Airspeed Challenge)

## Local Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

Deployed on [Railway](https://railway.app).
